2、接口自动化分类
	
	1）无关联多个用例自动化接口测试

3、接口选择
	
	理财类项目常见接口：
		
		1）登录接口    适合一类接口自动化
		
		2）注册接口    不适合
		
		3）购买理财产品  不适合
		
		4）赎回理财产品   不适合
		
		5）用户信息查询  适合
		
		6）产品信息查询   适合
		
		7）余额查询    适合
		
		8）充值    不适合
		
		9）提现    不适合
